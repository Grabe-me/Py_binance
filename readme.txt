
    Собственные движения цены ETH/USDT можно определить, рассматривая изменение цены ETH/USDT
    в отрыве от цены BTC/USDT. Для этого можно использовать следующий метод:

    Определите коэффициент корреляции между ценами ETH/USDT и BTC/USDT на заданном временном интервале.
    Для этого можно использовать статистические инструменты, такие как корреляционный анализ.

    Используя полученный коэффициент корреляции, вычислите изменение цены ETH/USDT,
    которое можно объяснить влиянием цены BTC/USDT.
    Для этого умножьте изменение цены BTC/USDT на коэффициент корреляции.

    Отнимите полученное значение из общего изменения цены ETH/USDT на заданном временном интервале.
    Это даст вам изменение цены ETH/USDT, которое можно приписать только собственным движениям
    цены ETH/USDT, исключив влияние цены BTC/USDT.
